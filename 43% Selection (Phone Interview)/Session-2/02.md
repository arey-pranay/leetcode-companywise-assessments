# Intuition

A **Binary Search Tree (BST)** maintains the property:

* All values in the **left subtree** are **less than** the node value.
* All values in the **right subtree** are **greater than** the node value.

To insert a new value:

* Start from the root.
* Compare the value with the current node.
* Move **left** if the value is smaller.
* Move **right** if the value is larger.
* When we reach a `null` position, insert the new node there.

Recursion makes this process simple by naturally traversing down the tree.

---

# Approach

1. If the root is `null`, create and return a new node with the given value.
2. Compare `val` with `root.val`.
3. If `val < root.val`:

   * If `root.left` exists, recursively insert in the left subtree.
   * Otherwise create a new node as `root.left`.
4. If `val > root.val`:

   * If `root.right` exists, recursively insert in the right subtree.
   * Otherwise create a new node as `root.right`.
5. Return the root after insertion.

This maintains the **BST property** while inserting the node at the correct position.

---

# Complexity

* **Time complexity:**
  O(h)
  where `h` is the height of the BST.

* Worst case (skewed tree): **O(n)**

* Best case (balanced tree): **O(log n)**

* **Space complexity:**
  O(h)
  due to recursive call stack.

# Code
```java []
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public TreeNode insertIntoBST(TreeNode root, int val) {
        if(root == null) return new TreeNode(val);
        if(root.left == null && root.right ==null){
            if(val > root.val) root.right = new TreeNode(val);
            else root.left = new TreeNode(val);
            return root;
        }
        if(val<root.val){
            if(root.left !=null )  insertIntoBST(root.left,val);
            else root.left = new TreeNode(val);
        }
        if(val>root.val){
            if(root.right !=null )  insertIntoBST(root.right,val);
            else root.right = new TreeNode(val);
        }
        return root;
    }
}
```
