## Intuition

In **postorder traversal**, we visit **all children first** and **then the node itself**.

For an **N-ary tree**, each node can have multiple children. So the idea is:

1. Traverse each child recursively.
2. After visiting all children, add the current node value to the result list.

This naturally fits a **recursive Depth-First Search (DFS)** approach.

---

## Approach

1. Create an empty list `ans` to store the traversal.
2. If the root is `null`, return the empty list.
3. Use a helper recursive function `func(node, ans)`:

   * If the node has children, recursively call the function for each child.
   * After all children are processed, add the current node's value to `ans`.
4. Return the result list.

### Steps during recursion

* Visit all children from **left to right**.
* After visiting children, **add the node value** to the list.

This ensures **postorder traversal: children → node**.

---

## Complexity

* **Time complexity:**
  (O(n))
  Each node in the tree is visited exactly once.

* **Space complexity:**
  (O(h)) recursion stack, where `h` is the height of the tree.
  In the worst case (skewed tree), it can be **O(n)**.

---

## Code

```java
/*
// Definition for a Node.
class Node {
    public int val;
    public List<Node> children;

    public Node() {}

    public Node(int _val) {
        val = _val;
    }

    public Node(int _val, List<Node> _children) {
        val = _val;
        children = _children;
    }
}
*/

class Solution {
    public List<Integer> postorder(Node root) {
        List<Integer> ans = new ArrayList<>();
        if(root == null) return ans;
        func(root, ans);
        return ans;
    }

    public void func(Node root, List<Integer> ans){
        if(root == null) return;

        for(Node child : root.children){
            func(child, ans);
        }

        ans.add(root.val);
    }
}
```
